# Minesweeper
### Description de projet
Ce projet consiste à créer le jeu Démineur en Rust et de créer un algorithme capable de résoudre efficacement le jeu sans jamais échouer peu importe la difficulté du niveau.

Il est possible de jouer au jeu et de lancer l'algorithme pour résoudre le restant des mines.

### Lancer le jeu
```Rust
cargo run
```

### Dépendances 
*  https://iced.rs/ (UI)
